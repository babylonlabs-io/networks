# `v1` Software Upgrade

## Upgrade overview

- **Upgrade version**: `v1.0.0-rc.7`
- **Upgrade height**: `200`

## Upgrade process

To upgrade your Babylon node, please perform the following steps:
- Stop your Babylon node
- Obtain the `v1.0.0-rc.7` binary. You can achieve this in multiple ways:
  - Download the binary from the [releases page](https://github.com/babylonlabs-io/babylon/releases/tag/v1.0.0-rc.7)
  - Build the binary on your machine
    ```shell
    git checkout v1.0.0-rc.7
    BABYLON_BUILD_OPTIONS="testnet" make install
    ```
  - If you’re working with Docker images, you can pull the pre-built Docker image:
    ```shell
    docker pull babylonlabs/babylond:v1.0.0-rc.7-mock-mainnet
    ```
- Start your Babylon node
